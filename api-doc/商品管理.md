## GET 商品列表

GET /api/group/goods/list

### 请求参数

|名称|位置|类型|必选|说明|
|---|---|---|---|---|
|cate_id|query|string| 否 |商品分类id|
|search|query|string| 否 |搜索|
|is_hot|query|string| 否 |是否热门（0：否，1：是）|
|is_new|query|array[string]| 否 |是否新品（0：否，1：是）|
|is_recommend|query|string| 否 |是否推荐（0：否，1：是）|
|sort|query|string| 否 |排序字段（sort，）|
|sort_order|query|string| 否 |排序方式（asc：升序，desc：降序）|
|Authori-Zation|header|string| 否 |none|

> 返回示例

> 200 Response

```json
{
  "status": 0,
  "msg": "string",
  "data": {
    "goodsList": [
      {
        "id": 0,
        "cate_id": 0,
        "title": "string",
        "image": "string",
        "min_price": "string",
        "max_price": "string",
        "fake_sales": 0
      }
    ],
    "count": 0
  }
}
```
