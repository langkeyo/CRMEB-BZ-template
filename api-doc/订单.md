## POST 创建订单

POST /api/group/order/createOrder

> Body 请求参数

```yaml
goods: '[{"product_id":11,"quantity":2},{"product_id":12,"quantity":2}]'
coupon_id: "1"

```

### 请求参数

|名称|位置|类型|必选|说明|
|---|---|---|---|---|
|Authori-Zation|header|string| 否 |none|
|body|body|object| 否 |none|
|» goods|body|string| 否 |商品数据|
|» coupon_id|body|string| 否 |优惠卷id|

> 返回示例

> 200 Response

```json
{}
```
