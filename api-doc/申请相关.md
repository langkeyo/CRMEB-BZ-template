# 申请相关

## POST 居间服务申请

POST /api/group/intermediary/apply

用户申请成为中介

> Body 请求参数

```yaml
name: 张三
phone: "13800138000"
sum: "50000.00"
notes: 我想申请成为中介，有丰富的销售经验

```

### 请求参数

|名称|位置|类型|必选|说明|
|---|---|---|---|---|
|Authori-Zation|header|string| 否 |none|
|body|body|object| 否 |none|
|» name|body|string| 是 |姓名|
|» phone|body|string| 是 |手机号码|
|» sum|body|string| 是 |金额|
|» notes|body|string| 是 |备注|

> 返回示例

> 请求成功

```json
{
  "status": 200,
  "msg": "申请成功"
}
```

```json
{
  "status": 400,
  "msg": "居间服务已申请,请勿重复申请"
}
```

```json
{
  "status": 400,
  "msg": "姓名不能为空"
}
```

> 401 Response

```json
{
  "status": 110002,
  "msg": "请登录"
}
```

### 返回结果

|状态码|状态码含义|说明|数据模型|
|---|---|---|---|
|200|[OK](https://tools.ietf.org/html/rfc7231#section-6.3.1)|请求成功|Inline|
|401|[Unauthorized](https://tools.ietf.org/html/rfc7235#section-3.1)|未授权|[UnauthorizedResponse](#schemaunauthorizedresponse)|

### 返回数据结构

状态码 **401**

|名称|类型|必选|约束|中文名|说明|
|---|---|---|---|---|---|
|» status|integer|false|none||none|
|» msg|string|false|none||none|

## POST 供应商申请

POST /api/group/supplier_apply

用户申请成为供应商

> Body 请求参数

```yaml
name: 王五
phone: "13800138002"
goods_name: 优质水果
business_license: /uploads/business_license.jpg
goods_images: /uploads/goods1.jpg,/uploads/goods2.jpg
hygienic_license: /uploads/hygienic_license.jpg
identity_card_r: /uploads/id_front.jpg
identity_card_g: /uploads/id_back.jpg
source: 自有果园
intro: 我们是专业的水果供应商，有自己的果园，提供优质新鲜水果

```

### 请求参数

|名称|位置|类型|必选|说明|
|---|---|---|---|---|
|Authori-Zation|header|string| 否 |none|
|body|body|object| 否 |none|
|» name|body|string| 是 |联系人姓名|
|» phone|body|string| 是 |联系人电话|
|» goods_name|body|string| 是 |商品名称|
|» business_license|body|string| 是 |营业执照照片路径|
|» goods_images|body|string| 是 |商品图片路径（多个用逗号分隔）|
|» hygienic_license|body|string| 是 |卫生许可证图片路径|
|» identity_card_r|body|string| 是 |身份证人像面图片路径|
|» identity_card_g|body|string| 是 |身份证国徽面图片路径|
|» source|body|string| 是 |产品来源商|
|» intro|body|string| 是 |商品介绍|

> 返回示例

## GET 查询申请状态

GET /api/group/apply/status

根据申请类型查询用户的申请状态和详细信息

### 请求参数

|名称|位置|类型|必选|说明|
|---|---|---|---|---|
|type|query|string| 是 |申请类型|
|Authori-Zation|header|string| 否 |none|

#### 枚举值

|属性|值|
|---|---|
|type|intermediary|
|type|leader|
|type|supplier|

> 返回示例

> 请求成功

```json
{
  "status": 200,
  "msg": "success",
  "data": {
    "type": "intermediary",
    "type_text": "中介申请",
    "has_apply": true,
    "apply_info": {
      "id": 1,
      "name": "张三",
      "phone": "13800138000",
      "sum": "50000.00",
      "notes": "我想申请成为中介，有丰富的销售经验",
      "status": 0,
      "status_text": "待审核",
      "create_time": "2025-08-01 10:30:00",
      "examine_time": null
    }
  }
}
```

